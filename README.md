# Gianni Cottone – Portfolio Website

Personal portfolio built with Next.js and Tailwind CSS to showcase projects, technical interests, and ongoing development.

🔗 Live: https://giannicottone.dev  
📦 Repo: https://github.com/giannicottone/gianni-portfolio

## Tech Stack

- **Framework:** Next.js 
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Deployment:** Vercel

## Project Status

This site is under active development. 
The current focus is on establishing a clean layout and maintainable structure before visual polish.

## Roadmap

This project is being built incrementally, with structure prioritized before polish.

### Phase One (Current)
- Finalize global layout and spacing
- Improve semantic HTML and accessibility
- Clean up unused components and styles

### Phase Two
- Flesh out portfolio content (projects, descriptions, links)
- Add dedicated project pages with technical context
- Refine navigation and page hierarchy

### Phase Three
- Visual polish and subtle UI enhancements
- Performance and SEO improvements
- Optional dynamic features as the site matures


## Workflow
- Development is iterative and focused on steady progress.
- Work is done directly on main
- Changes are committed and pushed in small, focused increments
- Emphasis is placed on clarity, maintainability, and correctness over strict process

This repository reflects an ongoing effort to balance clean structure, practical engineering decisions, and gradual refinement.
